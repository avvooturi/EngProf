Project 2 - Hardware Security with AI and Reconfigurability

Posted on: November 5, 2024

Category: Tech | Year: 2024

Problem Statement

    Threat Modeling ensures that security considerations are not an afterthought but a foundational aspect of design.

    Despite their extensive utility, existing threat modeling tools from various sources, e.g., Microsoft, MITRE have several limitations – (1) insufficient standardization, (2) complexity, extensive customization, and domain-specific knowledge associated, (3) static in nature, and (4) lack of root-cause analysis.

Solution

    ThreatGuard, using natural language processing (NLP) and formal property, develop an automated threat modeling tool that takes assets, attack surfaces, and target adversary models as inputs to generate a comprehensive model.

    We develop threat models for National Vulnerability Databases (NVD), e.g., CWE, CVE, CAPEC and train a LLM using fine-tuning and retrieval augmented generation to customize it for the given architecture.
